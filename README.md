# NLA-project_materials
Yusuf's chapter one research proposal on preexisting datasets. 
The raw NLA_datsets (2007-2022) were downloaded directly from the EPA website: https://www.epa.gov/national-aquatic-resource-surveys/data-national-aquatic-resource-surveys
See few metadata below: 
1- for PhytoplanKTON DATA
"ABUNDANCE"	"Number of organisms counted from sample"					
"ALGAL_GROUP"	"Broad algal group, used in metric calculation"	
"BIOVOLUME"	"Volume of cells per mL of sample"	
"DATE_COL"	"Date of sample collection"			
"DENSITY"	"Number of organisms/mL"					
"IS_DISTINCT"	"Distinct taxa classification per sample"					
"PSTL_CODE"	"State two letter postal code for site sampled"					
"PUBLICATION_DATE"	"Date the dataset was published to the NARS site"					
"SAMPLE_TYPE"	"Code identifying sample type"					
"SITE_ID"	"The NLA Site ID  for the study and year sampled"					
"TARGET_TAXON"	"Name of taxon at target taxon level"					
"TAXA_ID"	"NLA Taxon ID number"					
"UID"	"Unique identifier for the site/visit"					
"UNIQUE_ID"	"Unique site ID based on location that does not depend on year sampled or any other factor"					
"VISIT_NO"	"Sequential number of visit to site"
